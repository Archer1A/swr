# 更新镜像的概要信息<a name="swr_02_0032"></a>

## 功能介绍<a name="section14905762191056"></a>

## 请求消息<a name="section3270966102931"></a>

```
-H "Authorization: Bearer KEyEdpK..." \
-F "repository=busybox" \  
-F "category=linux" \  
-F "description=this is a busybox repository" \  
-F "is_public=true"
```
